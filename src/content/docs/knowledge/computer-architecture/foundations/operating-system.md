## 핵심 개념

운영체제는 시스템 소프트웨어의 핵심 구성요소 중 하나로, 사용자 프로그램과 하드웨어 사이의 인터페이스 역할을 한다. 주요 기능으로는 기본 입출력 처리, 저장장치와 메모리 할당, 그리고 동시에 여러 애플리케이션이 컴퓨터를 사용할 수 있도록 보호된 공유(protected sharing)를 제공하는 것이 있다. 대표적인 운영체제로는 Linux, iOS, Android, Windows가 있다. 운영체제는 I/O 장치 접근, 메모리 할당 등의 저수준 시스템 기능을 캡슐화하여 애플리케이션 프로그래머가 이러한 세부 사항을 직접 다루지 않아도 되게 한다.

## 예시

운영체제의 핵심 기능:
```
1. 입출력(I/O) 관리: 키보드 입력, 화면 출력 등
2. 메모리 관리: 프로그램에 메모리 할당 및 해제
3. 프로세스 관리: 여러 프로그램의 동시 실행 관리
4. 파일 시스템: 데이터의 영구 저장 관리
5. 보안: 프로그램 간 보호된 접근 제공
```
