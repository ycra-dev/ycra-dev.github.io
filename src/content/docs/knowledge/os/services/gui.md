## 핵심 개념

명령을 직접 입력하는 대신, 마우스 기반 윈도우-메뉴 시스템을 사용하여 운영체제와 상호작용하는 사용자 친화적 인터페이스.

## 데스크톱 메타포

**데스크톱 메타포**(desktop metaphor)란 화면(데스크톱)에 프로그램, 파일, 디렉토리, 시스템 기능을 나타내는 이미지나 **아이콘**(icons)을 표시하는 방식이다.

마우스를 사용하여 포인터를 이미지 위에 위치시키고, 마우스 버튼 클릭으로:
- 프로그램 호출
- 파일이나 디렉토리(**폴더(folder)**) 선택
- 명령이 포함된 메뉴 풀다운

## GUI의 역사

| 시기 | 이벤트 |
|------|--------|
| 1970년대 초 | Xerox PARC 연구소에서 GUI 연구 |
| 1973년 | Xerox Alto 컴퓨터에 최초의 GUI 등장 |
| 1980년대 | Apple Macintosh로 GUI 대중화 |
| 이후 | macOS의 **Aqua** 인터페이스, Windows 버전별 GUI 발전 |

## UNIX 시스템의 GUI

- 전통적으로 CLI 중심이었으나 다양한 GUI 사용 가능
- 오픈소스 프로젝트: **K Desktop Environment (KDE)**, GNU 프로젝트의 **GNOME** 데스크톱
- Linux와 다양한 UNIX 시스템에서 실행 가능

## 트레이드오프

- **장점**:
  - 사용하기 쉽고 직관적
  - 시각적으로 정보 표현
- **단점**:
  - 일부 시스템 기능은 GUI로 제공되지 않을 수 있음
  - 반복 작업에서는 CLI가 더 효율적
  - 시스템 리소스 소비가 CLI보다 큼
