## 핵심 개념

커널은 컴퓨터에서 항상 실행되고 있는 하나의 프로그램이다. 운영체제의 핵심으로서 항상 메모리에 상주하며 시스템과 사용자에게 서비스를 제공한다.

> 커널은 OS의 "심장"과 같은 핵심 엔진이다.

## 왜 필요한가

"운영체제"라는 용어가 다양한 역할과 기능을 포괄하기 때문에, 핵심 부분을 구분할 필요가 있다.

| 구분 | 설명 |
|------|------|
| **커널** | OS의 핵심, 항상 메모리에 상주하며 실행 |
| 시스템 프로그램 | OS와 연관되어 있지만 반드시 커널의 일부는 아님 |
| 응용 프로그램 | 시스템 운영과 연관되지 않은 모든 프로그램 |

## 동작 원리

1. 시스템 부팅 시 부트스트랩 프로그램이 커널을 메모리에 로드
2. 커널 실행 시작
3. 시스템과 사용자에게 서비스 제공 시작

## 모바일 OS에서의 커널

모바일 운영체제(iOS, Android)는 커널뿐만 아니라 **미들웨어**도 포함한다.

- **미들웨어**: 데이터베이스, 멀티미디어, 그래픽 등을 지원하는 소프트웨어 프레임워크
