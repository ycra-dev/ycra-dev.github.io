## 핵심 개념

프로그래밍 언어마다 적용 가능한 범위가 크게 다르다.

높은 적응성을 가진 언어들:
- **C#**: Windows, Mac, Linux, iOS, Android, 웹, 임베디드 시스템까지 거의 모든 플랫폼에서 사용 가능
- **JavaScript**: 웹 프론트엔드, 백엔드(Node.js), 모바일(React Native), 심지어 Arduino 보드와 로보틱스까지 가능
- **Python**: 웹 개발, 데이터 과학, 인공지능, 자동화, 스크립팅 등 다양한 분야

제한적 적응성을 가진 언어들:
- **R**: 통계 분석과 데이터 시각화에 특화
- **Go**: 시스템 프로그래밍과 서버 개발에 주로 사용

## 동작 원리

오늘 웹 개발을 하고 내일 Android 개발을 하고 싶다면, 또는 다양한 플랫폼과 기술에 관여하고 싶다면, 배우려는 언어의 적응성을 고려하는 것이 현명하다.

그러나 언어 적응성이 유일한 결정 요인이 되어서는 안 된다. 어떤 언어를 선택하든 **핵심은 하나를 골라서 충분히 숙련될 때까지 꾸준히 학습**하는 것이다.

## 예시

```
C# 하나로 개발 가능한 프로젝트:
├── 웹 애플리케이션 (ASP.NET Core)
├── 데스크톱 앱 - Windows (WPF/WinForms)
├── iOS 모바일 앱 (Xamarin.iOS)
├── Android 모바일 앱 (Xamarin.Android)
├── 게임 개발 (Unity)
├── 클라우드 서비스 (Azure Functions)
└── IoT/임베디드 (.NET nanoFramework)

JavaScript의 적응성 예시:
├── 웹 프론트엔드 (React, Vue, Angular)
├── 웹 백엔드 (Node.js, Express)
├── 모바일 앱 (React Native)
├── 데스크톱 앱 (Electron)
└── 서버리스 함수 (AWS Lambda)
```
