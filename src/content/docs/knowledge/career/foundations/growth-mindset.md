## 핵심 개념

Carol Dweck의 연구에 따르면 사람들은 두 가지 마인드셋 중 하나를 채택한다.

| 마인드셋 | 믿음 | 반응 |
|---|---|---|
| **고정 마인드셋** (Fixed) | 지능은 타고나는 것 | 실패를 능력 부족의 증거로 해석, 도전 회피 |
| **성장 마인드셋** (Growth) | 지능은 노력으로 발전 | 실패를 학습 기회로 인식, 도전 수용 |

엔지니어에게 성장 마인드셋은 **효과성에 대한 투자 의지를 결정짓는 전제조건**이다.

## 동작 원리

**성장 마인드셋의 실험 결과:**
- 홍콩대: 성장 마인드셋 학생의 73%가 보충 수업 등록 vs 고정 마인드셋 13%
- 뉴욕 7학년: 지능의 성장 가능성을 배운 학생들의 수학 성적 향상

**"Own Your Story" 마인드셋:**
자신의 이야기를 남이 정의하게 두지 않고 직접 써나가는 것이 핵심이다. 이론적 배경만 가진 상태에서도 새로운 분야를 학습하여 2년 만에 완전히 다른 전문가가 될 수 있다.

**실천 방법:**
- "나는 이걸 못해" → "아직 못하지만 배울 수 있어"로 바꾸기
- 불편한 상황에 자신을 의도적으로 노출하기

## 예시

```python
def approach_challenge(challenge, mindset):
    if mindset == "fixed":
        # "나는 이걸 못해" → 포기
        return "avoid"
    elif mindset == "growth":
        # "아직 못하지만 배울 수 있어" → 도전
        learn(challenge)
        practice(challenge)
        return "improve"
```

사례: 내성적인 엔지니어가 1년간 모든 소셜 이벤트에 참석 → 대화 기술 향상 + 귀중한 인맥 구축. 이것이 성장 마인드셋의 실천이다.
