## 핵심 개념

인간의 뇌는 뉴런과 시냅스의 네트워크로 구성되어 있으며, **학습이란 새로운 시냅스 연결을 형성하거나 기존 연결을 강화하는 과정**이다.

하나의 개념을 여러 감각 경로를 통해 접하면 더 많은 시냅스 연결이 형성되어 기억이 더 강하고 오래 지속된다.

**핵심 통찰**: 가르치는 것은 배우는 것의 가장 높은 단계다.

## 동작 원리

**다감각(Multisensory) 학습이 효과적인 이유:**
- 책으로만 읽기 → 시냅스 연결이 약함
- 코드 작성 + 에러 해결 + 설명하기 → 시냅스 연결이 강하고 다양함

**도제(Apprenticeship) 방식이 강의보다 효과적인 이유:**
- 전통적 강의: 청각(강의)과 시각(슬라이드)에만 의존
- 도제 방식: 직접 체험으로 모든 감각을 동원

**가르치기가 최고의 학습인 이유:**
1. 다른 사람에게 설명하려면 자신의 이해를 명확하게 정리해야 한다.
2. 질문에 답하면서 자신이 몰랐던 빈틈을 발견한다.
3. 팀의 역량을 높이면서 자신의 전문성도 심화된다.

## 예시

**단감각 학습:** 자바스크립트 문법 책을 읽기만 하는 것 → 시냅스 연결이 약함

**다감각 학습:** 튜토리얼을 따라 코드를 작성하고, 에러를 디버깅하고, 블로그에 정리하고, 동료에게 설명하는 것 → 시냅스 연결이 강하고 다양함

**도제 방식:** 시니어 개발자와 페어 프로그래밍을 하면서 실제 프로덕션 코드를 작성하는 것
