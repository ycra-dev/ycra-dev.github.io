## 핵심 개념

목표를 설정하는 것만으로는 충분하지 않다. 정기적으로 목표를 추적하고 업데이트해야 한다. 수 마일이나 잘못된 방향으로 간 후에야 실수를 발견하는 것은 큰 낭비이다.

목표 추적의 핵심은 반성(reflection)이다. 작은 기간과 큰 기간 모두에서 달성한 것을 되돌아보며, 적절한 진전을 이루고 있는지 혹은 조정이 필요한지 파악해야 한다.

## 동작 원리

추적 주기:
- **주간 리뷰**: 이번 주의 목표 달성 여부를 점검하고, 다음 주를 계획한다
- **월간 리뷰**: 월간 목표의 진행 상황을 평가하고 조정한다
- **분기별 리뷰**: 큰 방향성이 맞는지 점검한다
- **연간 리뷰**: 큰 목표와의 정렬 상태를 확인하고, 필요시 큰 목표 자체를 수정한다

## 예시

주간 리뷰 템플릿:

```
## 주간 목표 리뷰 (2026-W09)

### 달성한 것
- [x] Clean Code 챕터 5-7 완독
- [x] LeetCode 문제 5개 풀기
- [ ] 블로그 포스트 1개 작성 (80% 완료)

### 다음 주 조정사항
- 블로그 포스트 완성을 최우선으로
- 독서 목표를 2챕터로 축소 (현실적 조정)

### 큰 목표와의 정렬
- 시니어 개발자 목표 방향과 일치함 ✓
```
