## 핵심 개념

지식 피라미드는 특정 기술 분야에서 자신의 수준을 시각화하는 3층 모델이다. Mark Richards와 Neal Ford가 제시했다.

```
┌──────────────────────────┐
│   내가 할 수 있는 것     │ ← 기술 깊이 (Depth)
├──────────────────────────┤
│ 존재는 알지만 자신 없는 것│ ← 기술 너비 (Breadth)
├──────────────────────────┤
│   모르는 줄도 모르는 것  │ ← 미지의 영역
└──────────────────────────┘
```

상단 층이 넓을수록 해당 기술에 능숙한 것이다.

## 동작 원리

**자기 평가 도구로 활용:**

1. **깊이 층 식별**: 직업 유지/획득에 강한 깊이가 필요한 기술 파악
2. **너비 층 점검**: 각 기술에서 알아야 할 부분 파악
3. **학습 실행**: 해당 영역을 집중 학습

학습을 통해 깊이 층이 확장되면, 자연스럽게 너비 층(알지만 자신 없는 것)도 확장된다. 이것이 미래 학습의 발판이 된다.

**실용적 팁:** 인증(certification), 나노학위 등이 깊이 층의 범위를 정하는 데 도움이 된다.

## 예시

Python 지식 피라미드 예시:

| 층 | 내용 |
|---|---|
| 깊이 (할 수 있음) | 기본 문법, 리스트/딕셔너리 조작, 함수 작성 |
| 너비 (알지만 자신 없음) | async/await, 데코레이터 패턴 존재 인지 |
| 미지 | C 확장 모듈 작성, GIL 내부 동작 등 |
