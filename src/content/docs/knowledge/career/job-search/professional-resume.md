## 핵심 개념

100달러짜리 제트스키 대여 서비스도 풀컬러 전문 브로셔로 광고하는데, 연봉 6만 달러 이상의 가치를 가진 소프트웨어 개발자가 타이포가 가득한 Word 문서로 자신을 홍보하는 것은 모순이다.

이력서는 자신이라는 상품을 판매하는 **마케팅 도구**다.

## 동작 원리

**전문 이력서 작성자 고용 추천 이유:**
- 이력서 작성은 커리어에서 몇 번만 사용하는 스킬이다
- 전문가는 이미 수천 번의 경험을 가지고 있다
- CEO가 직접 코딩하지 않듯, 전문 분야가 아닌 것은 전문가에게 맡기는 것이 합리적이다
- 300~500달러 투자는 연봉 2~3% 인상으로 첫해에 회수된다

**전문 이력서 작성자 선택 기준:**
- IT 업계에 익숙한 사람
- 샘플 이력서를 보여줄 수 있는 사람

**직접 작성할 경우:**
- 온라인 버전 제작 (웹 개발자는 필수)
- 행동-결과(action-result) 언어 사용
- 반드시 교정 진행

## 예시

행동-결과 언어 변환:
```
[일반적인 문구]
"5년간 Java 개발 경험"

[행동-결과 언어]
"마이크로서비스 아키텍처 전환을 주도하여
 배포 시간을 70% 단축하고 시스템 가용성을 99.9%로 향상"
```

창의적 이력서: 비디오 게임 프로그래머가 자신의 이력서를 실제 플레이 가능한 비디오 게임으로 만든 사례.
