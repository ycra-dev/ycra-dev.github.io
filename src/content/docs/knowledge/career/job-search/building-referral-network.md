## 핵심 개념

대다수의 채용은 개인 추천을 통해 이루어진다. 추천을 통해 면접에 가면, 추천인의 사회적 신뢰가 부분적으로 전이되어 면접관이 이미 호감을 가지고 시작한다.

네트워크 구축은 취업이 필요하기 **전에** 시작해야 한다. 급할 때 만드는 네트워크는 효과가 없다.

## 동작 원리

네트워크 구축 5가지 방법:
1. **블로그 댓글**: 목표 회사 직원들의 블로그를 찾아 가치 있는 댓글을 남긴다
2. **로컬 유저 그룹**: 주기적으로 참여하고, 발표를 하면 빠르게 관계를 구축할 수 있다
3. **LinkedIn 직접 연락**: 커피 한 잔을 제안하되, 반드시 당신이 대접한다
4. **인터뷰/팟캐스트**: 블로그나 팟캐스트를 위한 인터뷰 요청은 자연스러운 연결 고리가 된다
5. **간접 연결**: 직접 모르는 사람도 공통 지인을 통해 소개받을 수 있다

중요한 태도:
- 진심으로 관계를 맺어라. 이용하려는 목적이 보이면 역효과가 난다
- 상대방에게 어떤 가치를 줄 수 있는지 먼저 생각하라
- "연결"이 아닌 "우정"을 만들어라

## 예시

12개월 네트워크 구축 계획:
```
월 1-3: 목표 회사 5곳 선정, 각 회사 개발자 블로그 팔로우 시작
월 4-6: 로컬 개발자 밋업 참석 (월 2회), 블로그에 정기 댓글
월 7-9: 밋업에서 발표 1회, LinkedIn으로 관심 회사
        개발자 3명과 커피 미팅
월 10-12: 관계가 형성된 회사에 채용 공고 모니터링,
          추천 요청 가능 상태

결과: 원하는 회사에 추천을 통해 지원 가능
```
