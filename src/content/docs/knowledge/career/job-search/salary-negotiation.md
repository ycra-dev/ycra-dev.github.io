## 핵심 개념

연봉 협상은 입사 지원 전부터 시작되며, 개인 브랜드와 평판이 협상력의 가장 큰 결정 요소다.

핵심 규칙: **먼저 숫자를 제시하는 쪽이 불리하다.**

자신의 희망 연봉을 먼저 말하면 상한선이 정해져 올라갈 수 없지만, 고용주가 먼저 말하면 거기서 올릴 수 있다.

## 동작 원리

협상력에 영향을 미치는 3가지 요소 (중요도 순):

1. **개인 브랜드와 평판**: 이름이 알려질수록 협상력이 커진다. 블로그, 팟캐스트, 강연, 오픈소스 기여로 구축
2. **입사 경로**: 공개 채용 지원(최악) → 개인 추천(좋음) → 회사가 먼저 찾아옴(최고). 니즈가 큰 쪽이 협상에서 불리하다
3. **숫자 먼저 말하지 않기**: 예산이 80,000~100,000달러인데 70,000달러를 먼저 제시하면 25,000달러 손해

카운터 오퍼 전략:
- 오퍼를 받으면 거의 항상 카운터해야 한다
- 배짱이 허락하는 한 높게, 최소 두 번까지 카운터한다
- 여러 회사에서 동시에 오퍼를 받으면 walk away 가능 → 협상력 극대화

## 예시

희망 연봉을 물어볼 때:
```
"전체 보상 패키지에 따라 다릅니다"
→ "예산 범위를 알려주시면 맞는지 확인하겠습니다"
```

현재 연봉을 물어볼 때:
```
"높으면 탈락할까 봐, 낮으면 손해볼까 봐 말씀드리기 어렵습니다"
```

카운터 오퍼:
```
"이 회사에서 일하고 싶습니다.
 X달러로 해주시면 오늘 확답 드리겠습니다"
```
