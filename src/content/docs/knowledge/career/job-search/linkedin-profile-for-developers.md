## 핵심 개념

LinkedIn은 리크루터가 후보를 발굴하는 가장 중요한 플랫폼이다. 빈약한 프로필은 기회를 놓치고, 최적화된 프로필은 잠자는 동안에도 인바운드 리드를 만든다.

## 동작 원리

프로필 최적화 핵심 요소:

| 요소 | 중요성 | 최적화 방법 |
|------|--------|------------|
| 헤드라인 | 최고 | "Senior React Developer" 대신 "React 성능 최적화 전문가 | 로딩 시간 70% 단축 경험" |
| 사진 | 높음 | 프로페셔널, 밝은 배경, 얼굴이 선명하게 |
| 요약(About) | 높음 | 1인칭으로 스토리 형태, 연락 가능 여부 명시 |
| 경험 | 높음 | 이력서와 동일하게 성과 중심 작성 |
| 기술(Skills) | 중간 | 지원 직무와 관련 키워드 추가, 상호 추천 확보 |
| 활동 | 중간 | 주기적으로 기술 관련 게시물 작성/공유 |

**"Open to Work" 설정**: 현직 중이라도 비공개로 리크루터에게만 구직 상태 표시 가능.

## 예시

- 헤드라인: "풀스택 개발자 | Node.js + React | B2B SaaS 스타트업 4년 경험 | 오픈소스 기여자"
- 요약 첫 줄: "저는 복잡한 기술적 문제를 사용자가 사랑하는 제품으로 만드는 개발자입니다"
- 스킬 추천(Endorsements): 10개 이상 팀원들에게 추천 요청하기
- 게시물: "오늘 배운 React 최적화 기법" → 동료들의 '좋아요'와 댓글 → 알고리즘 노출 증가
