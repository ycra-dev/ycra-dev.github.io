## 핵심 개념

100개 회사에 이력서를 뿌리는 산탄총 방식 대신, 10개 목표 회사를 선정하여 그 회사 직원들과 관계를 쌓고 내부에서 추천받는 방식이다. 경험이 부족하거나 불리한 조건에 있을 때 특히 효과적이다.

## 동작 원리

단계별 접근:

```
1. 목표 회사 리스트 작성 (5-20개)
2. LinkedIn에서 해당 회사 직원 찾기
3. 직접 연결 요청 (개인화된 메시지)
4. 커피챗 또는 정보 인터뷰 요청
5. 가치 제공 (블로그 글 공유, 관심사 대화)
6. 공고가 올라오면 내부 추천 요청 또는 직접 지원
```

이 방식이 효과적인 이유:
- 내부 추천된 지원자는 지원자 풀에서 상단에 노출됨
- 이미 알고 있는 사람을 채용 담당자에게 연결하면 신뢰가 높음
- 회사 문화를 미리 파악하여 핏이 맞는 곳에만 지원

## 예시

- 2년간 입사를 원하는 회사의 모든 직원과 LinkedIn 연결 → 블로그/밋업을 통해 해당 분야 명성 쌓기 → 내부 추천으로 채용
- "저는 귀사의 React 성능 최적화 블로그 포스트를 읽고 팀 문화가 궁금해졌습니다. 30분 커피챗 가능할까요?"
- 채용 담당자보다 팀의 시니어 개발자에게 직접 연락하는 것이 더 효과적인 경우 多
