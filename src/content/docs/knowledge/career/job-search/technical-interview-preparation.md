## 핵심 개념

기술 면접 준비는 단기간에 될 수 있는 것이 아니다. 최소 2-3개월의 체계적인 준비가 필요하고, 꾸준히 코딩 문제를 풀어야 한다. 준비의 핵심은 "알고리즘을 외우는 것"이 아니라 "문제 패턴을 인식하는 능력"을 기르는 것이다.

## 동작 원리

기술 인터뷰 구성:

| 유형 | 내용 | 준비 방법 |
|------|------|----------|
| 알고리즘/DS | 배열, 트리, 그래프, DP | LeetCode, Blind 75 |
| 시스템 설계 | URL 단축기, 채팅 앱 설계 | 책: "Designing Data-Intensive Applications" |
| 기술 지식 | 언어/프레임워크 심층 질문 | 공식 문서, 동작 원리 이해 |
| 코드 리뷰 | 버그 찾기, 개선 제안 | 다양한 코드베이스 읽기 |

**스터디 플랜 (3개월):**
- 1개월: 자료구조 기초 + Easy 문제 50개
- 2개월: Medium 문제 100개 + 시스템 설계 기초
- 3개월: Hard 문제 + 모의 인터뷰 + 특정 기업 준비

## 예시

- 매일 LeetCode 1-2문제 + 풀이 설명 글쓰기
- Pramp.com이나 Interviewing.io로 실전 모의 인터뷰
- 특정 회사 준비: Glassdoor/Blind에서 최근 인터뷰 후기 수집
- 스터디 그룹: 서로 면접관 역할 바꿔가며 연습
