## 핵심 개념

기본급만 비교하면 잘못된 결정을 내릴 수 있다. 기본급 $100K인데 스톡옵션, 401(k) 매칭, 의료보험이 없는 회사 vs 기본급 $90K인데 RSU, 풀 베네핏, 자유로운 원격근무를 제공하는 회사 중 어느 쪽이 더 좋은가?

## 동작 원리

총 보상 패키지 구성요소:

| 구성요소 | 종류 |
|---------|------|
| 현금 보상 | 기본급, 성과 보너스, 사인온 보너스, 인플레이션 조정 |
| 주식 | RSU(제한 주식), 스톡옵션, ESPP(우리사주매입제도) |
| 은퇴 | 401(k) 회사 매칭, 연금 |
| 의료 | 의료/치과/시력/정신건강 보험 |
| 시간/유연성 | PTO 일수, 원격근무, 유연 근무시간, 육아휴직 |
| 개발 | 교육비, 컨퍼런스 참가, 자격증 비용 |
| 기타 | 교통비, 식사, 헬스장, 전화/인터넷 지원 |

**주식 주의사항:**
- 베스팅(vesting) 일정 확인: 통상 4년에 1년 cliff
- 스타트업 스톡옵션: 행사가격과 현재 가치 차이 확인
- RSU는 세금 대상 (받을 때 근로소득으로 처리)

## 예시

- 오퍼 비교: 회사 A ($120K 기본) vs 회사 B ($100K 기본 + $30K RSU/년 + 풀 베네핏) → B가 더 유리할 수 있음
- Levels.fyi: 빅테크 회사들의 총 보상 패키지를 레벨별로 공개
- 협상 시: "기본급 조정이 어렵다면 사인온 보너스나 주식 부여량으로 보완 가능할까요?"
