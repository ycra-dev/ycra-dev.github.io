## 핵심 개념

대부분의 멀티태스킹은 실제로는 빠른 작업 전환(task-switching)이며 생산성을 떨어뜨린다. 진정한 생산성 향상은 배칭(batching)과 진정한 멀티태스킹(두 가지를 동시에 수행)을 구분하여 적용하는 것에서 온다.

## 동작 원리

**작업 전환의 문제**: 우리가 멀티태스킹이라 생각하는 대부분은 작업 간 빠른 전환이다. 전환할 때마다 뇌가 다시 해당 작업에 몰입하는 데 시간이 걸려 생산성이 떨어진다.

**배칭(Batching)**: 여러 번에 나눠 하던 같은 유형의 작업을 한 번에 모아서 처리하는 방법.
- 배치 후보: 이메일 답변, 전화 통화, 버그 수정, 짧은 미팅

**진정한 멀티태스킹**: 정신적 작업과 신체적 작업을 결합하는 것. 두 가지 정신적 작업을 동시에 하는 것은 불가능하다.

## 예시

```
[비효율적 멀티태스킹]
이메일 알림마다 확인하며 코딩
→ 집중력이 계속 끊기고 생산성 저하

[배칭 적용]
이메일: 하루 1회 일괄 처리
→ 코딩 시간에 완전히 집중 가능

[진정한 멀티태스킹]
운동하면서 오디오북 듣기 → 효과적 (신체 + 정신)
글 쓰면서 오디오북 듣기 → 비효과적 (정신 + 정신)
```
