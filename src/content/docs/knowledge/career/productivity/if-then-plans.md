## 핵심 개념

If-Then 계획(구현 의도, Implementation Intentions)은 특정 상황(if)이 발생했을 때 특정 행동(then)을 수행하겠다고 미리 결정하는 심리 기법이다.

심리학자 Peter Gollwitzer의 연구: 언제/어디서/어떻게 할지 미리 정한 학생의 **71%**가 과제를 제출했지만, 그렇지 않은 학생은 **32%**만 제출했다.

## 동작 원리

미루기의 근본 원인은 초기 활성화 에너지(activation energy)에 대한 저항이다. 순간에는 단기적 편안함이 의사결정을 지배한다. 하지만 if-then 계획을 미리 세우면 상황(cue)과 행동이 자동으로 연결되어 의식적 의지력이 필요 없어진다.

```python
# If-Then 계획의 일상 적용
daily_if_then_plans = {
    "아침 (에너지 높음)": "정신적 노력이 큰 창의적 작업",
    "20분 여유 시간": "코드 리뷰 / 인터뷰 피드백 / 이메일",
    "회의 직후": "짧은 버그 조사",
    "점심 후": "코드 읽기 / 학습",
    "퇴근 전": "내일 우선순위 정리"
}
# 효과: 작업 완수 확률 ~2배 증가
```

**실용적 if-then 패턴:**
- "if 3시 회의 후라면, then 이 오래된 버그를 조사하겠다"
- "if 저녁 식사 후라면, then Android 개발 강의를 30분 수강하겠다"
- "if 다음 활동 전 20분 여유라면, then 짧은 코드 리뷰를 하겠다"

## 예시

다양한 연구에서 if-then 계획이 PSAT 공부, 다이어트, 금연, 대중교통 이용 등에서 목표 달성률을 유의미하게 높임을 확인했다.

개발자 적용 예시:
- 아침 커피를 내리는 동안 → 어제 작성한 코드 회고
- 점심 식사 직후 → 기술 아티클 1편 읽기
- 매주 월요일 아침 → 이번 주 우선순위 재설정
