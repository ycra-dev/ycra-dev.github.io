## 핵심 개념

직장에서의 8시간 근무 중 실제 생산적인 작업 시간은 약 4시간에 불과하다. 자영업이나 프리랜싱으로 전환할 때 이를 이해하지 못하면 비현실적인 기대와 좌절을 겪게 된다.

## 동작 원리

8시간 근무일 분석:
```
총 근무 시간: 8시간
  - 사교/잡담: 1시간
  - 이메일/회의 등 오버헤드: 2시간
  - 일반적 게으름: 1시간
────────────────────────
실제 작업 시간: 4시간
```

자영업에서의 추가 감소 요인:
- 집에서는 아무도 감시하지 않아 집중도가 더 낮아질 수 있다
- 4시간의 직장 근무와 4시간의 자영업 집중도는 다르다
- 조깅과 사자에게 쫓기며 달리는 것의 차이: 긴장감이 다르다

따라서 자영업 초기에는 하루 4시간의 진짜 생산적 작업을 목표로 설정하는 것이 현실적이다.

## 예시

- 현 직장에서 시간 추적을 시작해 실제 생산 시간을 먼저 파악한다
- 자영업 전환 후 기대치: 하루 8시간이 아닌 4~6시간 생산적 작업
- 점진적으로 하루 6시간 생산적 작업을 목표로 훈련한다
- RescueTime 같은 도구로 실제 시간 사용 패턴을 객관적으로 측정한다
