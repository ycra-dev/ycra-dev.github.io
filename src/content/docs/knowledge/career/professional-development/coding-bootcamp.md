## 핵심 개념

코딩 부트캠프는 대학과 독학 사이의 세 번째 경로다.

**장점:**
- **짧은 학습 기간**: 주 6일, 하루 10-12시간 집중 학습. 일반 직장에서 코딩 시간이 20% 미만인 점을 감안하면 몇 달 만에 수년치 경험에 상당하는 학습이 가능
- **높은 취업률**: 좋은 부트캠프는 90% 이상의 취업 배치율
- **저렴한 비용**: 무료~$20,000 범위로 대학 등록금 대비 훨씬 저렴
- **완전 몰입**: 외국어의 이머전(immersion) 학습법과 유사한 효과

**단점:**
- **막대한 시간 투입**: 직장을 그만두고 모든 것을 포기해야 할 수 있음
- **극도로 어려움**: 여유가 전혀 없는 빠른 진도
- **사기 부트캠프 존재**: 누구나 부트캠프를 열 수 있어 질 낮은 프로그램이 많음
- **CS 지식 부족**: 실용적 기술에만 집중하여 컴퓨터 과학 이론이 약할 수 있음

## 동작 원리

전략:
1. 이전 수강생과 대화하여 사기 여부 확인
2. 전액 저축하여 빚 없이 등록
3. 스케줄을 완전히 비우기
4. 강의 후에도 남아서 추가 학습 및 네트워킹
5. 반에서 상위 10% 목표
6. 입학 전에 기초를 미리 학습

## 예시

```
부트캠프 입학 준비 (등록 1개월 전):
- 기본 프로그래밍 개념 습득 (변수, 조건문, 반복문)
- 생활비 예산 확보 (3-6개월분)
- 모든 부수적 일정 정리
- Hello World 수준 이상의 프로그래밍 가능

부트캠프 중:
- 매일 10-12시간 집중 학습
- 강의 후 남아서 추가 학습
- 동료 및 강사와 적극 네트워킹
- 포트폴리오용 프로젝트 구축
```
