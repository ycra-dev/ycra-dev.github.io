## 핵심 개념

가장 효과적인 학습은 '놀이(play)'를 통한 능동적 학습이다. 읽기나 듣기 같은 수동적 학습보다, **직접 해보고 가르치는 것**이 훨씬 깊은 이해를 만든다.

"학습 스타일이 다르다"는 것은 신화(myth)다. 모든 사람은 실행과 가르침을 통해 가장 잘 배운다.

## 동작 원리

자전거 타는 법에 대한 책을 아무리 많이 읽어도, 실제로 타보기 전에는 배울 수 없다.

놀이의 힘: 동물 왕국 전체에서 새끼 동물들은 놀이를 통해 생존 기술을 배운다. 인간도 마찬가지다. 자연스러운 호기심과 탐구가 학습을 가속화한다.

능동적 학습 순환:
```
훑어보기 → 바로 실험 시작 → 질문 생성
    ↑                              ↓
가르치기 ← 재적용 ← 참고자료에서 답 탐색
```

## 예시

- 기술 서적을 처음부터 끝까지 읽는 대신: 먼저 훑어보고 바로 실험 시작 → 막히는 부분 기록 → 참고자료 읽기 → 재적용 반복
- 프로그래밍 언어 학습: 문법서를 다 읽기 전에 간단한 프로젝트를 시작하고 부딪히면서 배우기
- 흥미로운 카드 게임의 수천 장 카드를 암기하려 노력하지 않아도 자연스럽게 외우게 되는 것과 같은 원리
- 최종 단계로 배운 것을 다른 사람에게 가르치기 (블로그, 대화 등)
