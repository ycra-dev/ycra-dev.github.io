## 핵심 개념

효과적인 자기교육은 '놀이 → 학습 → 재적용 → 가르침'의 순환 과정을 따른다. 학교 교육 시스템은 이 자연스러운 학습 방식을 오히려 "가르쳐서 없애버린다."

## 동작 원리

4단계 자기교육 사이클:

1. **놀이(Play)**: 최소한의 지식으로 바로 실험 시작. 호기심이 자연스럽게 질문을 만든다
2. **학습(Learn)**: 생성된 질문에 대한 답을 참고자료에서 찾는다. 동기와 맥락이 있어 흡수력이 높다
3. **재적용(Reapply)**: 새로 배운 지식을 다시 놀이에 적용. 새로운 영역 탐구, 새로운 질문 발견
4. **가르침(Teach)**: 배운 것을 자신만의 언어로 정리하여 다른 사람에게 전달. 이해를 시멘트처럼 굳힌다

```
놀이 → 질문 발생 → 참고자료에서 답 탐색
  ↑                              ↓
재적용 ←───────────── 가르침
```

## 예시

새 프레임워크 학습:
```
1. README만 읽고 바로 토이 프로젝트 시작 (놀이)
2. 막히는 부분 기록
3. 공식 문서에서 해답 탐색 (학습)
4. 다시 프로젝트에 적용 (재적용)
5. 블로그 포스트 작성 (가르침)
```

- 배우자나 동료에게 배운 것을 설명하는 것만으로도 가르침의 효과를 얻을 수 있다
- 학교 교육은 "먼저 이론, 나중에 실습" 순서를 강요하지만, 자연스러운 학습은 그 반대다
