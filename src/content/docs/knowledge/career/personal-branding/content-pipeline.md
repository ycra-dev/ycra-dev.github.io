## 핵심 개념

대부분의 초보 콘텐츠 크리에이터가 실패하는 이유는 계획 없이 즉흥적으로 콘텐츠를 만들기 때문이다.

콘텐츠 파이프라인의 첫 단계: **약 30개의 콘텐츠 아이디어 목록을 미리 만든다.** 그러면 "다음에 뭘 만들지?" 고민하는 시간이 사라진다.

## 동작 원리

제작 프로세스 문서화:
```
주제 선택
→ 스크립트/개요 작성
→ 녹화/작성
→ 편집
→ 업로드
→ 제목/설명/태그 작성
→ 공개 일정 설정
→ SNS 공유
```

이 프로세스를 문서화하면:
- 단계를 빠뜨리지 않는다
- 속도가 빨라진다
- 나중에 일부 작업(편집, 업로드 등)을 외주화할 수 있다

## 예시

- 30개 영상 아이디어 목록을 미리 작성 → 제작 시 즉시 시작 가능
- 카메라 스킬: 많이 할수록 좋아진다 — 완벽하지 않아도 일단 공개하라
- 프로세스 문서화로 일부 작업을 외주 가능 (편집, 자막 등)
