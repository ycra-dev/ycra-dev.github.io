## 핵심 개념

데이터 마이닝(Data Mining)은 대규모 데이터 집합에서 통계적 기법과 알고리즘을 활용하여 숨겨진 패턴, 상관관계, 유용한 정보를 추출하는 기술이다. "데이터에서 금을 캐낸다"는 의미의 비유로, 현대 디지털 경제의 핵심 기술이다.

## 동작 원리

**데이터 수집 방식**:
- 온라인 활동: 웹 검색 기록, 구매 내역, 소셜 미디어 활동
- 오프라인 활동: 신용카드 사용, 의료 기록, 교통 카드
- IoT 기기: 스마트폰 위치, 스마트 가전 사용 패턴
- 공공 기록: 부동산 거래, 법원 기록, 유권자 등록

**데이터 연결(Data Aggregation)**: 데이터 마이닝의 진정한 위력은 **여러 출처의 데이터를 결합**할 때 나타난다. 이름, 주소, 이메일 등의 식별 정보를 기준으로 서로 다른 데이터베이스를 연결하면 매우 상세한 개인 프로필이 구축된다.

**활용 분야**:
- **상업적 활용**: 고객 세분화, 추천 시스템, 사기 탐지, 신용 평가
- **정부 감시**: 테러 용의자 식별, 범죄 예측, 세금 사기 탐지
- **과학 연구**: 유전체 분석, 기후 데이터 분석, 역학 조사

**프라이버시 문제**:
- 개인이 동의하지 않은 방식으로 데이터가 활용될 수 있음
- 익명화된 데이터도 다른 데이터와 결합하면 재식별 가능
- 데이터 브로커(Acxiom, Experian 등)가 수억 명의 개인정보를 거래
- NSA의 대규모 감시 프로그램(에드워드 스노든 폭로)이 대표적 사례

## 예시

추천 시스템 (Amazon):
```
수집 데이터:
- 사용자 A가 구매한 책: Python 입문, 알고리즘, 자료구조
- 사용자 B가 구매한 책: Python 입문, 알고리즘, 머신러닝

데이터 마이닝 결과:
- A와 B의 구매 패턴이 유사
- B가 구매한 "머신러닝"을 A에게 추천
→ "이 상품을 구매한 고객이 함께 구매한 상품"
```

데이터 결합으로 프로필 구축:
```
소스 1 (온라인 쇼핑): 김철수, 서울, 전자기기 구매 이력
소스 2 (신용카드): 김철수, 서울, 고급 레스토랑 결제 이력
소스 3 (SNS): 김철수, 서울, 여행 사진, 관심사
소스 4 (위치 데이터): 김철수, 강남구 출퇴근 패턴

→ 결합 프로필: 30대, 서울 강남구 거주, 기술직, 고소득,
   맛집 탐방 취미, 연 2회 해외여행, Apple 제품 선호
→ 정밀한 광고 타겟팅에 활용
```
