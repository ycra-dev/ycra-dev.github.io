## 핵심 개념

### 구조와 동작

튜링 기계는 **제어 유닛**과 양방향으로 무한한 **테이프**로 구성된다:
- 테이프는 셀(cell)로 나뉘며, 각 셀에는 하나의 기호가 저장됨
- 어느 시점에서든 유한 개의 비공백 기호만 존재
- 제어 유닛은 현재 셀의 기호를 읽고 쓸 수 있으며, 좌우로 이동 가능

각 단계에서 현재 상태가 s이고 읽은 기호가 x일 때, f(s, x) = (s', x', d)이면:
1. 상태를 s'로 전환
2. 현재 셀에 x'를 기록 (x를 덮어씀)
3. d = R이면 오른쪽, d = L이면 왼쪽으로 한 칸 이동

f(s, x)가 정의되지 않은 경우(부분 함수이므로) 기계는 **정지(halt)**한다.

### 5-튜플 표현

전이 규칙은 (s, x, s', x', d) 형태의 5-튜플로 표현된다.
- 예: (s0, 0, s1, 1, R)는 "상태 s0에서 0을 읽으면 상태 s1로 전환, 1을 쓰고 오른쪽 이동"

### 집합 인식

튜링 기계 T가 문자열 x를 인식하려면, x를 테이프에 쓰고 초기 위치에서 시작했을 때 최종 상태(어떤 5-튜플에도 첫 번째 요소로 나타나지 않는 상태)에서 정지해야 한다.

- 정규 집합: 항상 오른쪽으로만 이동하는 튜링 기계로 인식 가능
- 비정규 집합 (예: {0^n 1^n | n >= 1}): 마커(M)를 사용하여 좌우로 왕복하며 인식

### 함수 계산

튜링 기계를 함수 계산에 사용할 때, 비음수 정수 n은 n+1개의 1로 표현(단항 표현)하고, k-튜플 (n1, n2, ..., nk)는 각 수의 단항 표현을 *로 구분하여 표현한다.

### 변형과 동치

다양한 변형이 존재하지만 모두 동일한 계산 능력을 가진다:
- 다중 테이프 튜링 기계
- 2차원 테이프
- 단방향 무한 테이프
- 비결정적 튜링 기계
- 범용 튜링 기계 (다른 모든 튜링 기계를 시뮬레이션)

## 예시

**연속된 11을 00으로 바꾸는 튜링 기계**:
```
5-튜플:
(s0, 0, s0, 0, R)  -- 0을 만나면 그대로 두고 오른쪽
(s0, 1, s1, 1, R)  -- 1을 만나면 기억하고 오른쪽
(s0, B, s3, B, R)  -- 공백이면 정지
(s1, 0, s0, 0, R)  -- 1 다음에 0이면 연속 1 아님
(s1, 1, s2, 0, L)  -- 1 다음에 1이면 두 번째 1을 0으로
(s1, B, s3, B, R)  -- 1 다음에 공백이면 정지
(s2, 1, s3, 0, R)  -- 첫 번째 1도 0으로 바꾸고 정지

테이프 입력: ...B 0 1 0 1 1 B...
실행: s0에서 시작 -> 0,1,0 읽고 이동 -> 1,1 발견 시 00으로 변환
최종 테이프: ...B 0 1 0 0 0 B...
```

**{0^n 1^n | n >= 1} 인식 튜링 기계** (마커 M 사용):
```
전략: 가장 왼쪽 0을 M으로, 가장 오른쪽 1을 M으로 반복 교체
000111 -> M00111 -> M0011M -> MM011M -> MM01MM -> MMM1MM -> MMMMMM (인식!)
```

**덧셈 함수 f(n1, n2) = n1 + n2**:
```
입력: (n1+1)개의 1, *, (n2+1)개의 1
5-튜플: (s0, 1, s1, B, R)  -- 첫 1 삭제
        (s1, *, s3, B, R)  -- n1=0이면 *삭제 후 정지
        (s1, 1, s2, B, R)  -- 다음 1도 삭제
        (s2, 1, s2, 1, R)  -- 나머지 1 건너뜀
        (s2, *, s3, 1, R)  -- *를 1로 교체

예: 2 + 3 => 111*1111 => B1B1111 => BBB11111 (결과: 5+1=6개의 1 -> 값 5)
```
