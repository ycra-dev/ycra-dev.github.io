## 핵심 개념

대규모 프로젝트에서 애자일을 적용할 때는 스케일 업(scale-up)과 스케일 아웃(scale-out)의 두 가지 관점이 있다. 스케일 업은 대규모 소프트웨어 시스템 개발에 애자일을 적용하는 것이고, 스케일 아웃은 조직 전체에 애자일을 도입하는 것이다. 대규모 프로젝트에서는 사전 설계, 시스템 문서화, 크로스팀 커뮤니케이션 메커니즘 등이 추가로 필요하다. SAFe(Scaled Agile Framework)는 대표적인 대규모 애자일 프레임워크이다.

## 예시

SAFe의 계층 구조: 포트폴리오 레벨(전략적 방향) → 프로그램 레벨(ART: Agile Release Train, 8-12주 PI) → 팀 레벨(2주 스프린트). 각 레벨에서 서로 다른 수준의 계획과 동기화가 이루어진다.
