## 핵심 개념

Googleyness는 Google이 강한 리더십을 나타내고 "겸손, 존중, 신뢰"를 구현하는 속성과 행동을 명시적으로 정의한 rubric이다. 처음에는 모호하게 사용되던 용어를 무의식적 편향의 위험을 인식한 후 공식화했다.

## 동작 원리

초기에 "Googley"라는 용어는 비공식적 문화 적합성 테스트로 사용되었으나, 모든 직원에게 다른 의미를 가지면서 "나와 비슷한 사람"이라는 무의식적 편향의 원천이 될 위험이 있었다.

Google이 명시적으로 정의한 Googleyness 기준:

| 속성 | 설명 |
|------|------|
| 모호함 속에서 번성 | 상충되는 메시지나 방향 속에서 합의를 형성하고 진전을 이룸 |
| 피드백을 가치 있게 여김 | 겸손하게 피드백을 주고 받으며, 피드백의 가치를 이해함 |
| 현 상태에 도전 | 저항이나 관성이 있어도 야심찬 목표를 설정하고 추구함 |
| 사용자를 최우선시 | 사용자에 대한 공감과 존중으로 최선의 이익을 추구함 |
| 팀을 배려 | 동료에 대한 공감과 존중으로 적극적으로 도움 |
| 올바른 일을 함 | 모든 일에 강한 윤리 의식을 가지고 어려운 결정도 기꺼이 내림 |

## 예시

면접관이 후보자와 "맥주를 한잔하고 싶은지"는 유효한 시그널이 아니다. 다양한 배경과 경험을 가진 사람을 채용해야 하므로, "나와 비슷한 사람"이 아닌 명시적 행동 기준으로 평가해야 한다.
