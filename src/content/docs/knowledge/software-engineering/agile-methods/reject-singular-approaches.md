## 핵심 개념

단일 접근법 거부(Reject Singular Approaches)란 기술 분야의 불평등을 해결하기 위해 단일 철학이나 방법론에 의존하지 말아야 한다는 원칙이다. 문제가 복잡하고 다요인적이므로 해결책도 다면적이어야 한다.

## 동작 원리

기술 산업에서 대표적인 단일 접근법의 오류: "인력 대표성 부족은 채용 파이프라인을 고치면 해결된다." 이것은 기본적 단계이지만 즉각적 해결이 필요한 문제는 아니다.

승진과 유지에서의 체계적 불평등을 인식하면서, 동시에 더 대표적인 채용과 인종/성별/사회경제적/이민 지위에 따른 교육 격차에 초점을 맞춰야 한다.

전체 문제 공간을 완전히 이해하는 것이 해결 방법을 결정하는 데 결정적이다 — 이는 중요 데이터 마이그레이션에서도, 대표적 인력 채용에서도 마찬가지이다.

## 예시

여성 엔지니어를 더 많이 채용하고자 할 때:
- 파이프라인 구축에만 집중하지 말 것
- 채용, 유지, 승진 생태계가 여성에게 얼마나 포용적인지 평가할 것
- 다양한 팀을 관리한다면, 심리적 안전에 집중하고 새 팀원이 환영받는다고 느끼도록 다문화 역량 투자를 할 것
