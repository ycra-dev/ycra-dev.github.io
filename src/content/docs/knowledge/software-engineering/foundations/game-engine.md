## 핵심 개념

현대의 복잡한 게임들은 대부분 자체 엔진을 만들지 않고 기존 게임 엔진을 사용하여 개발한다. 게임 엔진은 렌더링, 물리, 입력, 오디오, AI 등 공통 기능을 제공하여 개발자가 게임 콘텐츠에 집중할 수 있게 한다.

대표적인 게임 엔진:
- **Unity 3D**: C# 사용, 2D/3D 지원, 가장 많은 플랫폼 지원, 인디 개발자에게 인기
- **Unreal Engine**: C++ 사용, 고품질 그래픽, AAA 게임 스튜디오에서 선호

게임 개발은 경쟁이 극도로 치열하고, 긴 근무 시간과 상대적으로 낮은 보수가 특징이므로 **진정한 열정이 필요한 분야**다.

## 동작 원리

게임 개발자에게 C/C++ 능력이 매우 중요한 이유: 비디오 게임은 항상 최첨단 하드웨어의 한계를 밀어붙이기 때문이다. 또한 행렬 변환 등 복잡한 수학적 계산 능력도 필수적이다.

## 예시

```
게임 개발 경력 경로:
1. 대형 스튜디오: 안정적 급여, 팀의 일부로 특정 영역 담당
2. 인디 개발자: 창작 자유, 하지만 높은 리스크
3. VR 게임: 새로운 성장 분야, 기존 3D 게임 기술 활용 가능

핵심 기술:
- 게임 엔진 (Unity/Unreal)
- C/C++ (성능 최적화)
- 선형대수/행렬 변환 (3D 수학)
- 물리 시뮬레이션
```
