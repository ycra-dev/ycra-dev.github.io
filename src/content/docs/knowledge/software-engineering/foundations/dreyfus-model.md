## 핵심 개념

드레이퍼스 모델(Dreyfus Model of Skill Acquisition)은 학습자가 기술을 습득하는 과정을 5단계로 분류한 모델이다. Stuart와 Hubert Dreyfus가 1980년에 제시했다. 각 단계에서 필요한 학습 방법이 다르며, 한 분야의 전문가도 다른 분야에서는 초보자일 수 있다.

## 동작 원리

5단계 기술 습득 과정:
1. **초보자(Novice)**: 규칙에 의존. 맥락 없이 규칙을 따름
2. **고급 초보자(Advanced Beginner)**: 경험에서 패턴을 인식하기 시작
3. **능숙자(Competent)**: 문제를 체계적으로 분석. 의식적 계획 수립
4. **숙련자(Proficient)**: 직관적 이해. 상황을 전체적으로 파악
5. **전문가(Expert)**: 무의식적 능숙함. 규칙을 넘어선 직관적 판단

이 모델은 "Four Stages of Competence"와도 연결된다: 무의식적 무능력 → 의식적 무능력 → 의식적 능력 → 무의식적 능력.

## 예시

```
# 프로그래밍에서의 드레이퍼스 단계 예시

초보자: "for 루프는 이렇게 쓰는 거구나"
        → 문법 규칙을 기계적으로 적용

고급 초보자: "이런 패턴이 자주 나오네"
        → 반복되는 코드 패턴을 인식

능숙자: "이 문제는 이런 구조로 설계해야겠다"
        → 의식적으로 설계 결정을 내림

숙련자: "이 코드 뭔가 이상하다" (직관)
        → 전체적 맥락에서 문제를 포착

전문가: 복잡한 시스템을 자연스럽게 설계
        → 규칙을 의식하지 않고 직관적으로 판단
```
