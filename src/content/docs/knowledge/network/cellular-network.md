## 핵심 개념

셀룰러 네트워크는 서비스 지역을 '셀(cell)'이라는 작은 영역으로 분할하고 각 셀에 기지국을 배치하여, 제한된 주파수 스펙트럼을 **재사용**함으로써 대규모 무선 통신을 가능하게 하는 네트워크 구조이다.

## 동작 원리

서비스 지역을 벌집 모양의 셀로 나누고, 각 셀의 중심에 기지국(base station)을 배치한다. 핵심 아이디어는 **주파수 재사용**이다. 인접하지 않은 셀에서는 같은 주파수를 동시에 사용할 수 있으므로, 제한된 주파수 자원으로 많은 사용자를 수용할 수 있다.

**세대별 발전**:
- **1G (1980년대)**: 아날로그 음성 통화 전용
- **2G (1990년대)**: 디지털 전환. GSM, CDMA. 문자 메시지(SMS) 도입
- **3G (2000년대)**: 모바일 데이터 통신 본격화. 수 Mbps
- **4G/LTE (2010년대)**: 고속 데이터. 수십~수백 Mbps. 영상 스트리밍 가능
- **5G (2020년대)**: 초고속(수 Gbps), 초저지연(1ms 목표), 초연결

**핵심 기술 요소**:
- **핸드오프(Handoff)**: 사용자가 이동하며 한 셀에서 다른 셀로 넘어갈 때, 통화/연결이 끊기지 않도록 기지국 간 자동 전환
- **IMEI**: 15자리 숫자로 모든 휴대폰 기기를 고유하게 식별
- **SIM**: 가입자 정보를 담은 칩. 기기와 네트워크 가입을 분리

## 예시

셀 구조와 주파수 재사용 개념:

```
     [A]     [B]     [C]
  [C]     [A]     [B]     [C]
     [B]     [C]     [A]

A, B, C = 서로 다른 주파수 그룹
같은 문자의 셀은 같은 주파수를 재사용
인접 셀은 다른 주파수를 사용하여 간섭 방지
```

세대별 데이터 속도 비교:

| 세대 | 최대 속도 | 대표 서비스 |
|------|----------|-----------|
| 2G (GSM) | ~14.4 Kbps | 이메일, 텍스트 |
| 3G | ~2-21 Mbps | 웹 브라우징, 앱 |
| 4G/LTE | ~100-300 Mbps | 영상 스트리밍 |
| 5G | ~1-10 Gbps | AR/VR, IoT |
