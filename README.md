# ycra.dev

CS 지식 정리 및 개발 블로그

## Structure

Medallion Architecture 기반 지식 관리 시스템

- **TIL (Bronze)** - 시간 중심의 일일 학습 로그
- **Knowledge (Silver)** - 주제 중심의 원자적 지식 노트
- **Blog (Gold)** - 정제된 인사이트와 스토리텔링

## Tech Stack

- [Astro](https://astro.build/)
- [Starlight](https://starlight.astro.build/)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build
npm run build

# Preview build
npm run preview
```
